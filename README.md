# Design Patterns
The goal of this project is to acquire an understanding of design patterns and implement them in C++. The design patterns that I will be implementing are:

1) Builder
2) Factory
3) Prototype
4) Singleton
5) Adapter
6) Bridge
7) Composite
8) Decorator
9) Facade
10) Flyweight
11) Proxy
12) Chain of responsibility
13) Command
14) Interpreter
15) Iterator
16) Mediator
17) Memento
18) Observer
19) State
20) Strategy
21) Template Method
22) Visitor
